# React-ToDo-List
Practice with React. Make a ToDo list the utilizes much of what a basic React program has to offer.

In App.js, I put a lot of notes down in the comments so I could refer back to them when I do further react projects. Doing this project, I learned the following skills:

* `useState` for managing state
* Rendering lists with `.map()`
* Handling user events like clicks and typing
* Updating and filtering arrays in state
* Storing objects in state
* Applying dynamic styles based on state

Run my website by typing in `npm start` into your terminal.
